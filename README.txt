Turn your Answering Contact Form answers into a beautiful FAQ page.

== Description ==

Answering Contact Form FAQ Page Add-on is a add-on to the Answering Contact Form. This plugin will turn the questions and answers that you have added to the Answering Contact Form database into a beautiful FAQ page with jQuery toggle.
All you have to do is to paste the shortcode into a page of your choice and now you have a FAQ page.

For more information, demo and documentation  go to the official plugin site <a href="http://answeringcontactform.com" taget="_blank">answeringcontactform.com</a>.

== Installation ==
= Installation =
You need Answering Contact Form for this plugin to work properly.<br> 
1. Install Answering Contact Form FAQ Page Add-on via the plugin directory or by uploading the files manually to your server.<br> 
2. Paste the shortcode [acf_faq_page] into a page of your choice.<br> 
5. All the answers that you have added in Answering Contact Form->Answers should now be displayed.<br>
